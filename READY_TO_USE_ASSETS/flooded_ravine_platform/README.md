# flooded_ravine_platform

Low-poly, Blender-authored four-metre platform module for **Flooded Ravine**.

It includes the complete three-lane track bed/deck, sleepers, six rails, fasteners, filled terrain/water/catwalk shoulders, left/right mountain walls, and the biome ceiling or open ravine treatment. Runtime places one module every 4 m on the sampled track basis, so it follows curves and slopes.
